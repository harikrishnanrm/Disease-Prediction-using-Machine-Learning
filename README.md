# Disease-Prediction-using-Machine-Learning
This project focuses on developing a machine learning-based system to predict diseases based on user-provided symptoms. The model is designed to assist in early-stage disease identification by analyzing patterns in medical datasets. By leveraging data preprocessing techniques and machine learning algorithms, the system provides accurate and efficient predictions.
# Objective
The main objective of this project is to build a predictive model that can:
1. Identify possible diseases based on input symptoms
2. Assist in early diagnosis and decision-making
3. Demonstrate the application of machine learning in healthcare
# Features
Predicts diseases based on symptoms provided by the user.
Performs data preprocessing including cleaning and feature selection.
Supports model training and evaluation using medical datasets.
Provides fast and reliable prediction results.
Improves accuracy through optimization techniques.
# Technologies Used
**Programming Language**: Python
**Libraries**: scikit-learn, pandas, numpy
**Concepts**: Machine Learning, Data Preprocessing, Classification Algorithms
# Dataset
The model is trained using a medical dataset containing:
      List of symptoms
      Corresponding diseases
Data preprocessing steps include:
      Handling missing values
      Encoding categorical data
      Feature selection for better performance
#  Working Principle
The system takes symptoms as input from the user and processes the data through a trained machine learning model. The model analyzes patterns learned during training and predicts the most probable disease. The accuracy of the prediction is improved using preprocessing and optimization techniques.
# Model Implementation
Collected and preprocessed medical dataset
Split dataset into training and testing sets
Trained machine learning models using scikit-learn
Evaluated model performance using accuracy metrics
Optimized the model for better prediction results
# Results
Achieved reliable prediction accuracy on test data
Demonstrated effective classification of diseases based on symptoms
Showed practical application of machine learning in healthcare
